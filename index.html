Селектор * - селектор для всіх єлементів на сторінці

Селектор за назвою атрибута - викликається вказанням цього атрибуту наприклад a{} p{} та ін.

  • Комбінований селектор - викликається за допомогою указання наприклад класу та id
  • Дочірній селектор - викликається для всіх вкладених елементів, наприклад ul>li{}

Селектор за класом - викликається за допомогою .назва класу

Селектор за декількома класами - викликається за допомогою .назва класу.назва класу

Селектор за id - викликається за допомогою #назва id

Селектор за атрибутом - викликається допомогою [атрибут]

Псевдокласи які змінюють колір при наведенні створені за допомогою a:hover, a:active

Зміна кольору беспосередньо у HTML за допомогою style="color:колір ;"
